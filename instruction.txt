Запуск скрипта с 0:

Для использования скрипа необходимо скачать актуальную версию python. Есть 2 варианта как это сделать:
	Через Microsoft Store найдя его через поиск windows (значок лупы или win+S), после чего написать в поиске Microsoft Store "python 		3.10 compiler" и скачать "Python 3.10", затем установите программу.

	Через командную строку PowerShell. Для этого нажмите win+R, напишите в открывшейся строке "powershell" и нажмите клавишу ENTER. 		Далее в открывшемся окне введите эту команду
	Invoke-WebRequest -Uri https://www.python.org/ftp/python/3.10.0/python-3.10.0-amd64.exe -OutFile python-3.10.0-amd64.exe
	и нажмите ENTER. После этого в командной строке PowerShell напишите
	.\python-3.10.0-amd64.exe
	откроется программа для установки интерпритатора python, завершите процесс установки программы.

Скачайте и распакуйте архив со скриптом.
Откройте панель администратора, нажав клавиши win+R и написав в открывшемся поле "cmd", а затем нажав ENTER.
Перейдите в папку с распакованным скриптом при помощи команды "cd ПУТЬ В ПАПКУ СО СКРИПТОМ", заменив русский текст на путь к папке. Установите нужные програмные файлы командой:
pip install -r requirements.txt
или если не сработало, то:
python install -r requirements.txt
Напишите команду для запуска скрипта:
python main.py
Готово, скрипт работает. Для остановки скрипта в терминале нажмите Ctrl+C.


Когда скрипт начал работу, перейдите по ссылке "https://t.me/Yuyderbot" в бота и запустите его чтобы он запомнил вас и присылал вам сообщения об обновлениях в интересующих группах.


Настройка работы скрипта:
Для изменения списка групп удалите или добавьте ссылку на группу в файл groups.txt. Добавляйте группы через пробел.

Для изменения ключегого слова по которому проверяются посты измените файл keywords.txt.

Для того чтобы больше никто не смог пользоваться ботом кроме вас, после запуска бота измените значение в файле save_id_rule.txt на 0. Чтобы вернуть всё обратно измените значение на 1

Описание файлов:
В папке со скриптом находятся файлы "main.py", "bot.py", "instruction.txt", "README.md" и две папки "configuration_files" и "service_files". Файлы "main.py" и "bot.py" - это файлы с кодом проекта, их лучше не трогать, в файле "instruction.txt" находится эта инструкция, а в файле "README.md" описание проекта. Папку "service_files" тоже лучше не трогать - там находятся сервисныеи файлы скрипта. В папке "configuration_files" находятся файлы для корректировки работы скрипта.
Файл groups.txt нужен для добавления групп, которые вы хотите отслеживать. Просто введите ссылки на группы через пробел. Пример содержания файла:
https://vk.com/synergyuniversity https://vk.com/public218803811
Файл keywords.txt нужен для редактирования слов, по которым происходит поиск. В этом файле должны быть перечислены через  пробел слова которые будут являтся маркерами для отправки уведомления. Пример того что должно содержаться в файле:
машина купить продать каен
Если скрипт обнаружит одно из слов в посте, то отправит уведомление.
Файл save_id_rule.txt нужен для того чтобы была возможность закрыть или открыть достук к скрипту другим аккаунтам. При значении 1 ботом могут воспользоваться любые пользователи telegram, при значении 0 ботом могут пользоваться только те, кто уже запускал бота ранее.
